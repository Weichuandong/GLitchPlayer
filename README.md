### 简单的音视频播放器

#### 目标

- 通过FFmpeg解析视频文件（支持MP4格式）
- 通过FFmpeg解码
- 使用SDL2创建窗口画布
- 使用OpenGL进行视频渲染上屏
- 使用OpenAL进行音频播放
- 音视频同步处理

#### 技术栈

- FFmpeg：负责解封装，解码，格式转换等
- SDL2:负责窗口管理，事件处理（键盘/鼠标）等
- OpenGL：视频渲染，色彩空间转换等
- OpenAL：高级音频处理（3D音效，混音）等

#### 要求

- 使用CMake创建工程
- 除了SDL，FFmpeg，OpenGL，OpenAL，Freetype外，不引入其他第三方库
- 提供`暂停`，`重新播放`，`停止`功能
- 支持解析字幕
